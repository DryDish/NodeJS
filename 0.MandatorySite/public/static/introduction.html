Strings Strings

Since JavaScript does not allow for explicit type declaration we can do our best to make it know what we want.

One way to do it is to wrap it in the object we want it to be a type of.

We can check the type of the object with the method typeof() which returns a string of the type We can then use console.log() to print what it returns to check it. 


              // for String                 
              const string_const = String("jupiter")
              console.log(typeof(string_const));
              console.log(string_const);
              // The output is: 
              
                string
                jupiter
              

              // And for Number
              const number_const = Number("12")
              console.log(typeof(number_const ))
              console.log(number_const );
              // The output is: 
              
                number
                12

Even if we initialize the 12 with quotation marks it will still be cast to Number thanks to the wrapper

If we want to represent a number with two decimals we can achieve it in two ways. We can make it be a string, or we can use the .toFixed() method. 


              // String
              const number = "1.10";
              console.log(number);
              // Output:
              
                1.10
              
              // .toFixed()
              // Casting the string "1.10" to a number, thus removing the last 0
              const number2 = Number(number); 
              // Printing the number with two decimals with the toFixed() method
              // We pass the number 2 to indicate the amount of decimals
              console.log(number2.toFixed(2));
              
                1.10

In JavaScript Strings are essentially arrays of chars. Thanks to this fact, we can use indexes to retrieve a specific character. 


              const letters = "abc";
              // to grab the letter 'c' we can do the following:
              letters[2];
              
              // the issue is that this returns the letter,
              // so we best store it somewhere.
              const letter = letters[2];
              console.log(letter);
              // Output:
              
                "c"
Objects Objects

Objects in JavaScript are made by surrounding one or more key value pairs in a curly brace 


              const myObj = {"message": "Hello, earthling! I bring peace."};

We can assign additional values like this:


              myObj.type = "sms";
              console.log(myObj);
              // Output:
              
                {
                  message: 'Hello, earthling! I bring peace.',
                  type: 'sms'
                }

If we want to add a boolean as an attribute then we declare it true or false without quotation marks like this 


              myObj.isActive = true;
              console.log(myObj);
              // Output:
              
                {
                  message: 'Hello, earthling! I bring peace.',
                  type: 'sms',
                  isActive: true
                }

We can retrieve the specific attributes of objects by calling objectName.keyName 


              const value = myObj.message;
              console.log(value);
              // Output:
              
                'Hello, earthling! I bring peace.'

If we want to remove a property from an object we can simply call delete and the property name 


              delete myObj.isActive;
              console.log(value);
              // Output:
              
                {
                  message: 'Hello, earthling! I bring peace.',
                  type: 'sms'
                }
Arrays Arrays

To initialize an array in JavaScript you simply place your attributes in between two square brackets separated by a comma like this: 


                const letters = ["a","b","c", 1, 3, false, myObj];

To add values to an array you have to use the push() method


              let friend1 = {name:"peter", liar: false};
              let friend2 = {name: "Sandy", liar: false};
              let friend3 = {name: "Mandy", liar :true};

              friends.push(friend1);
              friends.push(friend2);
              friends.push(friend3);

              console.log("Printing the array of friends\n");
              console.dir(friends);
              // Output:
              
                Printing the array of friends

                [
                  { name: 'peter', liar: false },
                  { name: 'Sandy', liar: false },
                  { name: 'Mandy', liar: true }
                ]

To search the contents of an array you can use indexOf() it will return the index in the array where the first occurrence of the passed variable is, if it exists. If its not found then it will return -1 

            
              console.log("indexOf friend2:", friends.indexOf(friend2));
              // Output:
              
                indexOf friend2: 1

If you want to insert something into the array at a specific location you can use splice().
Splice is used to remove things from a specific index as well as potentially insert.

It's syntax is as follows:
splice(start: number(the starting index),
deleteCount: number(amount of items to delete(from starting index)),
...items (any amount of items to insert(optional)))

example: splice(index, numberToDelete, itemToInsert, itemToInsert2) 

              
                let friend4 = {name: "New Friend", liar: false};
                // inserting friend4 into the array's index 2 without removing
                friends.splice(2,0, friend4);
                console.log(friends);
                // Output:
                
                  [
                    { name: 'peter', liar: false },
                    { name: 'Sandy', liar: false },
                    { name: 'New Friend', liar: false },
                    { name: 'Mandy', liar: true }
                  ]

If you want to remove the last element of a list you can use the pop() method.
It removes and returns the last element of a list.
Here is an example of it in action: 

              
                console.log(friends);
                // Output:
                
                  [
                    { name: 'peter', liar: false },
                    { name: 'Sandy', liar: false },
                    { name: 'New Friend', liar: false },
                    { name: 'Mandy', liar: true }
                  ]
                
                friends.pop();
                console.log(friends);
                // Output:
                
                  [
                    { name: 'peter', liar: false },
                    { name: 'Sandy', liar: false },
                    { name: 'New Friend', liar: false }
                  ]
                
                const poppedFriend = friends.pop();
                console.log("Friend list:")
                console.log(friends);
                console.log("Popped friend:")
                console.log(poppedFriend);
                // Output:
                
                  Friend list:
                  [
                    { name: 'peter', liar: false },
                    { name: 'Sandy', liar: false }
                  ]
                  Popped friend:
                  { name: 'New Friend', liar: false }

You can also iterate through the items of an array with the use of the spread operator.
If you for example wanted to copy the contents of one array to another, the easiest way would be to do something like this 

              
                let fakeFriends = [...friends];

This will take one at a time, the contents of friends and place it in fakeFriends, this is different to
let fakeFriends = friends;
as this only makes fakeFriends a reference of friends, and not a new copy
